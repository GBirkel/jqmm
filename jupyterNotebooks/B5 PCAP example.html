B5: PCAP example¶

Introduction¶

This notebook provides a description of how to reproduce the results in Alonso-Gutierrez et al 2014 using the jQMM library.

Setup¶

First set environment and path variables properly:

In [1]:
quantmodelDir = '/users/hgmartin/libraries/quantmodel'

This is the only place where the jQMM library path needs to be set.

In [2]:
%matplotlib inline

import sys, os
pythonPath = quantmodelDir+"/code/PCAP"
if pythonPath not in sys.path:
    sys.path.append(pythonPath)
os.environ["QUANTMODELPATH"] = quantmodelDir

Then import needed modules:

In [3]:
import PCAPlib
import os
